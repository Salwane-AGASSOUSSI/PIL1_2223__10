Rapport de fin de projet

Informations générales

Nom du projet : Projet Intégrateur

Date de début : 8 juin 2023

Date de fin : 30 juin 2023

Objectifs du projet

Le projet Intégrateur avait pour objectif de développer une application web pour la gestion des emplois du temps dans un établisement scolaire.

Les principaux objectifs étaient :

  • Créer une interface conviviale pour les utilisateurs.
  • Permettre la création, la modification et la suppression des emplois du temps.
  • Implémenter un système de notifications pour les modifications.
  • Filtrer les données relatives aux emplois du temps.

Réalisations

Au cours du projet, les réalisations suivantes ont été accomplies :

  • Développement de l'application web utilisant les technologies HTML, CSS , JavaScript et django (python).
  • Mise en place d'une base de données MySQL pour stocker les informations des utilisateurs et des emplois du temps.
  • Intégration d'un système de notifications en temps réel.
  • Création de rapports statistiques à partir des données de l'application.

Résultats et bénéfices

Le projet Intégrateur a été livré dans les délais prévus et a atteint les objectifs fixés. Voici les principaux résultats bénéfiques obtenus :

  • L'apprentissage et l'appriation de git et GitHub.
  • L'apprentissage et l'utilisation de django pour la réalisation du projet.
  • Gestion du travail en groupe et des problèmes internes qui en découlent.
